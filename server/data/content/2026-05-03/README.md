## How to post

1. Open the per-draft `draft-N.md` brief.
2. Film or screen-record the shot list items.
3. Edit in CapCut or InShot — drop in the on-screen text at the listed timecodes.
4. Open TikTok / Instagram → upload the cut → search the suggested sound by name → attach.
5. Paste the platform-specific caption + hashtags from the brief.
6. Set the cover frame per the brief's advice.
7. Post at the listed best time.

> **Why no auto-post?** TikTok & IG require platform-licensed audio to be picked inside the native app — a third-party cannot attach trending sounds via API without losing the licensed-audio reach boost. Manual upload takes ~10 sec per post and gets meaningfully better distribution.
